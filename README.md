## Overview
AI UI Generator is a full-stack web application that generates UI layouts based on user prompts.

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js + Express
- AI Logic: Planner, Generator, Validator architecture
- Version Control: Git & GitHub

## Features
- Generate UI from text prompt
- Display structured UI plan
- Live HTML preview
- Prompt history memory
- Error handling
